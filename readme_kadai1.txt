kadai1.lzh の中身の説明

readme_kadai1.txt … このテキストファイル。
kaeru.uni … 「蛙の歌」のUNIファイル。
gosenfu.pdf … 音符を書き込んでいない五線譜。課題１−１で必要。
yorokobi.jpg … 「よろこびの歌」の楽譜の画像ファイル（jpg形式）

課題内容

課題１−１
　kaeru.uni をテキストエディタで開き、UNIデータを解析して楽譜に直せ。
　但し、４分音符をinterval値で120とする。
　楽譜は gosenfu.pdf を印刷し、これに音符を記入して提出すること。
　音符は声部１と声部２に分けて記入すること。

課題１−２
　yorokobi.jpg の楽譜を解析し、UNIデータに直せ。
　但し、４分音符をinterval値で120とする。
　UNIデータ作成にはテキストエディタを用いること。

＜提出方法＞
課題１−１は楽譜をスキャンまたはカメラで撮影し、そのファイルをmanabaに提出。
課題１−２についてはUNIファイルをmanabaに提出。
